HTML ENTITIES

  • &lt; represents the less-than sign <
  • &gt; represents the greater-than sign >
  • &amp; represents the ampersand &
  • &copy; represents the copyright symbol ©
  • &reg; represents the registered trademark symbol ®
  • &euro; represents the euro currency symbol €
  • &pound; represents the pound currency symbol £
  • &yen; represents the yen currency symbol ¥
  • &quot; represents the double quotation mark "
  • &apos; represents the single quotation mark '
  • &nbsp; represents a non-breaking space  
  • &mdash; represents an em dash —
  • &ndash; represents an en dash –
  • &lsquo; represents a left single quotation mark ‘
  • &rsquo; represents a right single quotation mark ’
  • &ldquo; represents a left double quotation mark “
  • &rdquo; represents a right double quotation mark ”
